🚀 TELEGRAM-TO-GOOGLE-SHEETS LIVE LEAD TRACKER - made this for personal use
========================================================================

A lightweight, 100% free automation tool built using Google Apps Script 
and the Telegram Bot API. Send client leads, data points, or tracking 
status updates directly from your mobile phone via Telegram, and instantly 
append them as structured rows inside your Google Spreadsheet.

3. 🛠️ SETUP GUIDE
------------------------------------------------------------------------

[Step 1: Create a Telegram Bot]
1. Search for @BotFather on Telegram and open a chat.
2. Send the command: /newbot
3. Give your bot a name (e.g., Lead Tracker Bot) and a unique username 
   ending in 'bot' (e.g., MyLeadDataBot).
4. Copy the HTTP API Token generated by BotFather. Keep this secure.

[Step 2: Set Up Google Sheets]
1. Create a fresh Google Spreadsheet.
2. Define your sheet columns across row 1 matching your workflow:
   * Column A: NAME
   * Column B: INSTA LINK
   * Column C: CONTACT NO.
   * Column D: STATUS
3. Click on "Extensions" in the top navigation bar, then select "Apps Script".

[Step 3: Add and Configure Code]
1. Erase any default boilerplate code inside the script window.
2. Paste the following configuration engine code block:

--- SCRIPT CODE START ---
const CONFIG = {
  TELEGRAM_TOKEN: "YOUR_TELEGRAM_BOT_TOKEN_HERE", 
  SHEET_NAME: "Sheet1"                            
};

function doPost(e) {
  try {
    const contents = JSON.parse(e.postData.contents);
    if (!contents.message || !contents.message.text) return;
    
    const chatId = contents.message.chat.id;
    const text = contents.message.text;
    
    if (text.toLowerCase() === "/start") {
      sendText(chatId, "👋 Bot is active!\n\nPlease send data using this exact comma format:\nName, Insta Link, Contact No, Status");
      return;
    }
    
    const dataArray = text.split(",").map(item => item.trim());
    if (dataArray.length < 4) {
      sendText(chatId, "❌ Formatting Error!\n\nPlease use the exact format:\nName, Insta Link, Contact No, Status");
      return;
    }
    
    const name = dataArray[0];
    const instaLink = dataArray[1];
    const contact = dataArray[2];
    const status = dataArray[3];
    
    const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheets()[0];
    sheet.appendRow([name, instaLink, contact, status]);
    
    sendText(chatId, "✅ Added '" + name + "' to the sheet successfully!");
  } catch(error) {
    Logger.log(error.toString());
  }
}

function sendText(chatId, text) {
  const url = "https://api.telegram.org/bot" + CONFIG.TELEGRAM_TOKEN + "/sendMessage";
  const payload = {
    "method": "post",
    "payload": {
      "chat_id": String(chatId),
      "text": text
    }
  };
  UrlFetchApp.fetch(url, payload);
}

function setWebhook() {
  const scriptUrl = ScriptApp.getService().getUrl();
  if (!scriptUrl || scriptUrl.indexOf("exec") === -1) {
    Logger.log("❌ Error: Deploy your script as a Web App first!");
    return;
  }
  const url = "https://api.telegram.org/bot" + CONFIG.TELEGRAM_TOKEN + "/setWebhook?url=" + scriptUrl;
  const response = UrlFetchApp.fetch(url);
  Logger.log("Result: " + response.getContentText());
}
--- SCRIPT CODE END ---

3. Modify the CONFIG constants block at the top with your credentials.
4. Click the Save icon (floppy disk logo).

[Step 4: Deploy as a Web App]
1. Click the Deploy button (upper-right corner) -> "New deployment".
2. Click the gear settings icon next to 'Select type' and pick "Web app".
3. Apply settings exactly:
   * Execute as: Me (your-email@gmail.com)
   * Who has access: Anyone
4. Select "Deploy" and authorize access using your Google Account if prompted.

[Step 5: Bind the Webhook Connection]
1. In the toolbar at the top of your Apps Script editor, toggle the active 
   function dropdown menu from 'doPost' to 'setWebhook'.
2. Click "Run".
3. Check the Execution log. If it outputs '{"ok":true,"result":true...}', 
   your live data binding is successful!

------------------------------------------------------------------------
5. 🔍 TROUBLESHOOTING & FAIL-SAFES
------------------------------------------------------------------------
* The Bot isn't responding to /start or inputs:
  Verify that 'Who has access' under deployment settings is set to 'Anyone' 
  instead of 'Only myself'.

* Re-running setWebhook returns a 404 Error:
  Your token in the code config block has a typo or is missing. Verify it 
  with @BotFather.

* Code modifications aren't registering on Telegram:
  Every code change requires a brand new version deployment. Go to Manage 
  Deployments -> Edit Pencil -> Version: New Version -> Deploy, then 
  re-run the setWebhook function.
